# 自动滚动表格库

如果你需要在网页中展示一张很长的表格，但是又不想让用户在页面上滚动，那么自动滚动表格库可以帮助你实现这个功能。

以下是三个常用的自动滚动表格库：

1. [jQuery.floatThead](https://github.com/mkoryak/floatThead) - 这个库使用 jQuery 实现。它可以让表头在页面滚动时一直保持在页面的顶部，同时也能让表格的其他部分在垂直方向上滚动。
2. [fixed-data-table](https://github.com/facebook/fixed-data-table) - 这个库是 Facebook 开发的一个 React 组件。它可以处理大型、高度可定制的表格，并且支持水平和垂直方向上的滚动。
3. [sticky-table-headers](https://github.com/jmosbech/StickyTableHeaders) - 这个库可以让表头在页面滚动时一直保持在页面的顶部，同时也能让表格的其他部分在垂直方向上滚动。它支持多种浏览器和移动设备。

以上是三个常用的自动滚动表格库，你可以根据自己的实际需求选择合适的库来使用。
